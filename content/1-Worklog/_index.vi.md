**Tổng quan:** Phần này giới thiệu worklog, tóm tắt các tuần hoàn thành, các công việc đã thực hiện và kết quả đạt được.  

Tôi đã hoàn thành chương trình thực tập trong vòng **12 tuần**, tập trung vào việc học và thực hành các dịch vụ AWS cơ bản, quản lý dữ liệu, bảo mật, ứng dụng serverless, phân tích và trực quan hóa dữ liệu. Dưới đây là tóm tắt công việc từng tuần:

**Tuần 1:** [Làm quen với AWS, tạo tài khoản Free Tier, thiết lập bảo mật, quản lý IAM và tạo Budget cơ bản](1.1-week1/)  

**Tuần 2:** [Thiết lập VPC, subnet, Internet Gateway, NAT Gateway, Security Group, EC2 instance, VPN site-to-site, Route53 hybrid DNS, VPC peering và Transit Gateway](1.2-week2/)  

**Tuần 3:** [Triển khai AWS Backup, tạo kế hoạch backup, thử nghiệm restore; S3 Bucket, Storage Gateway, triển khai website tĩnh với S3 & CloudFront, versioning, replication](1.3-week3/)  

**Tuần 4:** [Tạo S3 bucket, triển khai hạ tầng, backup plan & notifications, test restore; Virtual Machine import/export, Storage Gateway, file system Multi-AZ, test hiệu năng, website S3 nâng cao](1.4-week4/)  

**Tuần 5:** [Bảo mật với Security Hub, quản lý VPC, EC2 & Lambda với Tagging; IAM Users, Policies, Roles, KMS; CloudTrail & Athena; kiểm soát quyền hạn; dọn dẹp tài nguyên](1.5-week5/)  

**Tuần 6:** [Tạo VPC, EC2 & RDS Security Group, triển khai RDS database, deploy ứng dụng, backup/restore; kết nối EC2 qua RDP & Fleet Manager; cấu hình SQL Server & Oracle; tạo Migration Task; troubleshoot](1.6-week6/)  

**Tuần 7:** [Quản lý dữ liệu trên S3, DynamoDB, Redshift; pipeline dữ liệu với Kinesis, Glue, DataBrew, EMR; phân tích dữ liệu bằng Athena & Kinesis Data Analytics; trực quan hóa với QuickSight; ứng dụng serverless và dashboard tương tác; sử dụng CloudShell, SDK, Cloud9](1.7-week7/)

**Tuần 8:** [Xây dựng kiến trúc AWS cho dự án; thu thập yêu cầu, dựng Network Architecture Diagram, thiết kế VPC, subnet, routing, EC2, RDS, API Gateway, CloudFront, CI/CD và hoàn thiện sơ đồ theo feedback mentor](1.8-week8/)

**Tuần 9:** [Xây dựng bản nháp Proposal BDSS; viết Executive Summary, Problem Statement, Solution Overview; phân tích kiến trúc BDSS; tổng hợp nội dung từ proposal nhóm và chuẩn hóa theo template](1.9-week9/)

**Tuần 10:** [Xây dựng workshop dựa trên Proposal & Proposal Template; viết Introduction, Background, Objectives; xây dựng Architecture, Data Flow, CI/CD, Security; tổng hợp Cost & Risk để hoàn thiện Workshop Draft](1.10-week10/)

**Tuần 11:** [Tổng hợp nội dung từ 2 proposal; chuẩn hóa Executive Summary, Problem, Solution Architecture; xây dựng outline slide; chuẩn hóa toàn bộ nội dung sang tiếng Anh để chuẩn bị báo cáo cuối kỳ](1.11-week11/)

**Tuần 12:** [Trình bày dự án trước mentor; ghi nhận phản hồi về kiến trúc, CI/CD, cost, security; chỉnh sửa slide và tài liệu; tổng hợp action items để chuẩn bị cho báo cáo cuối kỳ](1.12-week12/)
