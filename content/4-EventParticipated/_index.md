### [Event 1](4.1-Event1/)
&emsp;**Event Name:** AWS Mastery #2 – CloudFormation & CDK Workshop  
&emsp;**Time:** November 17, 2025  
&emsp;**Location:** Online event (AWS Community Vietnam)  
&emsp;**Role:** Participant  

&emsp;**Event Description:**  
&emsp;The workshop focused on Infrastructure as Code (IaC) using CloudFormation and AWS CDK. Speakers introduced IaC concepts, explained the structure of CloudFormation templates, demonstrated commands such as `cdk deploy` and `cdk diff`, and guided participants on deploying infrastructure through code. Additional topics included Docker fundamentals, containerization, and services like ECS, EKS, and App Runner.

&emsp;**Value Gained:**  
&emsp;- A solid understanding of IaC workflows, drift detection, and CloudFormation best practices.  
&emsp;- Knowledge of CDK constructs (L1–L3) and automated infrastructure deployment.  
&emsp;- Practical insights into container orchestration on AWS.  
&emsp;- Improved ability to analyze and deploy DevOps pipelines using IaC.

### [Event 2](4.2-Event2/)
&emsp;**Event Name:** AWS Cloud Club – First Cloud AI Journey Workshop  
&emsp;**Time:** November 29, 2025  
&emsp;**Location:** Offline event for Cloud Clubs (Ho Chi Minh City)  
&emsp;**Role:** Participant  

&emsp;**Event Description:**  
&emsp;The workshop introduced the AWS Cloud Club community and provided a learning roadmap for Cloud + AI for students. Cloud Club Captains shared their AWS learning journeys, covering cloud fundamentals, AI/ML basics, GenAI concepts, study resources, certification paths, and practical career-development guidance.

&emsp;**Value Gained:**  
&emsp;- A clearer understanding of why cloud fundamentals are essential before diving into AI/GenAI.  
&emsp;- A structured AWS learning roadmap (Cloud Practitioner → Solutions Architect Associate).  
&emsp;- Effective self-study strategies and common mistakes beginners should avoid.  
&emsp;- Expanded professional network through connections with Cloud Club members and peers interested in cloud and AI.
